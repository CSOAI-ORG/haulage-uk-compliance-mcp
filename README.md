UK Operator Licence, tachograph, drivers' hours, DVSA roadside checks for UK haulage operators and 3.5T+ goods vehicles.

## Install

```bash
pip install haulage-uk-compliance-mcp
```

## Tools

| Tool | Purpose |
|------|---------|
| `check_operator_licence` | Verify Operator Licence requirement and current status by VRM or postcode |
| `calculate_drivers_hours` | EU + GB drivers' hours rules: daily/weekly driving + rest periods |
| `tachograph_audit` | Tachograph data analysis for compliance gaps |
| `vehicle_safety_check` | Walkaround check, gate check, and PMI compliance |
| `road_user_levy_check` | HGV Road User Levy / Direct Vision Standard / Clean Air Zone status |
